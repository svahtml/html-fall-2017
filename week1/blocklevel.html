Learning Block Level Elements

There are a number of ways to format text with HTML.

Paragraph Tag

The paragraph tag is use dot define large blocks of text within an article or on a page.

Headline Tags

Headline tages are use to define sections within an article. There are 6 version of the headline tag.

Break Tag

The break tag will add a line break within an element.

Example

Dan Hahn
June 7, 1944

My Great Company
111 5th Ave
New York NY 10003

Lists

List are a way to display a group of conent in a list form.

There are two types of list. Ordered and unordered.

Example

1. Apple

2. Orange

3. Grape

1. Snickers
2. Skittles
3. Milky Way

Ordered List

  1. Orange
  2. Grape
  3. Apple

Unordered List

  • Snickers
  • Skittles
  • Milky Way

Nested List

  1. Red
  2. Blue
    • Dark Blue
    • Sky Blue
  3. Green
